# django-customuser

## 概要
* djangoのカスタムユーザーモデルを使用
* username不使用→emailでログイン
* allauthは不使用

## 基本情報
* データベースはアップしてない(defaultのsqlliteで検証→mysqlに移行予定)
* プロジェクト名 cuproject
* アプリ名 usersapp
